# Basic Logical Connectives

## Conjunction, Disjunction and Negation
Let $p$ and $q$ be propositions.

| Logical Connective | Semantic Words | Logic Symbol |
| ---- | ---- | ---- |
| Conjunction | $p$ and $q$ | $p \wedge q$ |
| Disjunction | $p$ or $q$ | $p \vee q$ |
| Negation | not $p$ | $\neg p$ |

### Corresponding Semantic words and Logic Symbol
| Semantic Words | Logic Symbol |
| ---- | ---- |
| $p$ but $q$<br><br>$p$ also $q$<br><br>$p$ while $q$<br><br>$p$ yet $q$<br><br>$p$ although $q$<br><br>$p$ however $q$<br><br>$p$ nevertheless $q$<br> | $p \wedge q$ |
| not both $q$ and $q$ | $\neg (p \wedge q)$ |
| both not $p$ and and not $q$ | $\neg p \wedge \neg q$ |
| either $p$ or $q$ | $p \vee q$ |
| neither $p$ nor $q$ | $\neg (p \vee q)$ |
